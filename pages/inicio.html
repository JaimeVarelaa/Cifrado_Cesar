Primeros pasos cifrando

Estimadas/os estudiantes.

    Como pudimos ver en clase en cuanto el humano fue capas de generar información hubo interesados en robarla, por lo que se tuvieron que desarrollar técnicas para ocultarla, para este primer trabajo ustedes desarrollarán un programa el cual publicarán en la web (Pueden usar google) que les permita cifrar o descifrar una oración con método Cesar, bastando con que se le indique el módulo con que se trabajara. 

Rúbrica.

  Recuerden la rúbrica para la entrega. Se deberán entregar 1 enlace con la liga al programa y otro con la liga a la documentación.

Rúbrica de la Documentación

  • Portada 2 %

  • Índice 5  %

  • Introducción (5 %)  (Cada miembro del equipo deberá desarrollar y redactar su propia introducción) 

  • Objetivo ( 5% )

  • Desarrollo (70 % atención al desglose)

  1. Se deberá documentar el programa de manera segura (No se aceptaran impresiones por lo que usen su ingenio) (20%)
  2. Programa en web de cifrado y descifrado CESAR utilizando como base el código ascii y permitiendo al usuario seleccionar el módulo a utilizar, funcionando (Pueden usar las herramientas de google (50 %)

  • Conclusión (10 % (Cada miembro del equipo deberá desarrollar y redactar su propia introducción))

  • Bibliografía 3 %

Nota MUY IMPORTANTE: Equipos de 3 personas indispensable.
 
 Saludos y mucho éxito.
